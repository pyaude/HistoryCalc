# HistoryCalc — 需求文档

## 1. 产品概述

**HistoryCalc** 是一款面向普通大众的 Flutter 移动计算器应用。其核心差异化在于：**历史记录可被复用并参与当前计算**。用户可随时调出底部抽屉查看最近的计算记录，点击任意记录即可将其**结果值**插入到当前表达式中，而非开始新的计算。这让多步运算变得极为高效。

## 2. 目标用户

- 普通大众用户
- 对操作直觉性要求高，无需学习成本

## 3. 功能需求

### 3.1 计算引擎

| 功能 | 描述 |
|------|------|
| 基本四则运算 | 加法 `+`、减法 `-`、乘法 `×`、除法 `÷` |
| 平方根 | `√` 对当前数值开平方根 |
| 平方 | `x²` 对当前数值求平方 |
| 百分比 | `%` 百分比运算 |
| 正负号切换 | `±` 切换当前数值的正负 |
| 小数点 | `.` 支持小数输入 |
| 退格删除 | `⌫` 删除最后一个字符 |
| 清除 | `C` 清除当前输入；长按或二次点击变为 `AC` 全部清除 |
| 等号 | `=` 执行计算并得出结果 |
| 连续运算 | 按输入顺序依次计算（非运算符优先级） |

### 3.2 历史记录系统

- 保留最近 **10 条**计算记录
- 每条记录包含：**表达式字符串** + **计算结果**
- **FIFO 策略**：第 11 条记录自动移除最早的第 1 条
- **本机持久化**：使用本地存储（Hive / SharedPreferences），重启 App 后保留
- 底部抽屉面板展示，可随时打开/关闭

### 3.3 历史记录复用（核心差异化）

- 底部抽屉以列表展示历史记录（最近在上）
- 打开/关闭抽屉**不影响**当前正在进行的计算
- 点击某条记录 → **仅将其结果值**插入到当前表达式末尾
- 插入后抽屉自动关闭，用户可继续编辑并完成计算

### 3.4 UI/UX 设计

- **纯暗色主题**，沿用已有的 `AppTheme.darkTheme`
- 布局结构（从上到下）：
  - 表达式 + 结果显示区
  - 计算器按键区（4列网格）
  - 底部栏（含历史记录触发按钮）
- 底部抽屉：`DraggableScrollableSheet` 或 `BottomSheet`

## 4. 非功能需求

- 纯前端应用，无需后端/云端
- 无需用户系统
- 不包含科学计算（三角函数、对数等）
- 不包含亮色主题切换

## 5. 技术栈

| 层 | 技术 |
|----|------|
| 框架 | Flutter (SDK ^3.6.0) |
| 状态管理 | flutter_riverpod |
| 本地存储 | hive（推荐，性能好）或 shared_preferences |
| 字体 | google_fonts (Inter) |
| 平台 | Android + Web |
